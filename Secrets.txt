🔐 Required GitHub Secrets

Set these under:
Repo → Settings → Secrets and variables → Actions → New Repository Secrets

Name	                    Description

DOCKERHUB_TOKEN	          Docker Hub access token (generate via docker hub)
EC2_HOST	                Public IP or DNS of EC2 instance (better to add elastic ip to avoid ip change when restart)
EC2_SSH_KEY	              Private key for SSH to EC2  (generate key within the server or use relevant key)
